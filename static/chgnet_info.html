CHGNet Documentation

CHGNet (Crystal Hamiltonian Graph Neural Network)

What is CHGNet?

CHGNet is a state-of-the-art machine learning model that predicts structure, energy, forces, stress, and magnetic moments of crystalline materials. Trained on extensive density functional theory (DFT) calculation data from the Materials Project database, it achieves high accuracy predictions thousands of times faster than traditional DFT calculations.

Prediction Algorithm Flow

CHGNet employs a sophisticated graph neural network approach to materials prediction:

1
Graph Representation
Convert crystal structure into atomic graph with atoms as nodes and bonds as edges
2
Neural Network Processing
Learn complex interatomic interactions using graph neural networks
3
Multi-property Prediction
Simultaneously predict energy, forces, stress, and magnetic properties
4
Structure Optimization
Search for most stable structure through iterative optimization
